### ❓ 문제

---

0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.<br>


#### > &nbsp;입력

---

첫째 줄에 정수 `N(0 ≤ N ≤ 12)`이 주어진다.<br>


#### > &nbsp;출력

---

첫째 줄에 N!을 출력한다.<br>


### ✔️ 풀이

---

{% highlight python linenos %}

def f(x): return x * f(x - 1) if x > 1 else 1

print(f(int(input())))

{% endhighlight %}

재귀 함수를 이용해주었다.
